# FishTrack Africa - QR Sticker Branding

## 🎯 "Track the Bite" - Branded QR Code Sticker

Created: March 19, 2026

### Files

- **qr-sticker.html** - Interactive branded QR sticker generator
- **qr_base.png** - Base QR code (1000x1000px, links to fishtrack-sa.netlify.app)

### How to Use

1. **Open qr-sticker.html in a browser**
   - View the branded sticker design
   - Click "Download High-Res Sticker" for a 300 DPI print-ready PNG (900x900px)

2. **Print Settings**
   - **Resolution:** 300 DPI minimum
   - **Size:** 3" x 3" (75mm x 75mm) recommended
   - **Material:** Weatherproof vinyl sticker paper (outdoor use)
   - **Finish:** Matte or glossy (matte = less glare for QR scanning)

3. **Test Before Mass Print**
   - Print ONE test sticker first
   - Scan with multiple phones (Android + iPhone)
   - Check that it opens: https://fishtrack-sa.netlify.app
