## 📋 Usage Recommendations

### For Stickers/Posters:
- **Small (5cm x 5cm):** Use any design - QR will scan perfectly
- **Medium (10cm x 10cm):** Ideal for all designs
- **Large (20cm x 20cm):** Perfect for posters/banners

### Print Services:
1. **Online Printing (Sticker Mule, Vistaprint, etc.):**
   - Upload files directly
   - Select "Glossy" or "Matte" finish
   - Minimum order: 50-100 stickers (±R500-R1000)

2. **Local Printers:**
   - Provide files on USB
   - Request "full bleed" printing
   - Ask for UV-resistant ink for outdoor use

3. **At Home:**
   - Use photo paper or sticker paper
   - Print at "Best" quality setting
   - Recommended: HP, Canon, Epson photo printers
